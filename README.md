# Puzzling-Rooms
## Object Specifications
 - player is ball
 - Seesaw is object with pivot point
 - Puzzle ball is heavy object that can be pushed
 - End Hole is zone where if Puzzle is in End Hole the puzzle is solved

## Gameplay loop
 - Start screen asks amount of players (1 or 2)
 - On game start with # of players tell the player how to play with a short pause
 - Tell player(s) GO! and start a timer
 - On puzzle solved tell player(s) SOLVED! and show time
 - Next Puzzle button, Retry button, and Quit to Main Menu button.
